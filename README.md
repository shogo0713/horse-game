# 🐎 競馬ベッティングシミュレーター

React + TypeScript で実装した競馬ベッティングシミュレーションゲームです。  
オッズに基づく重み付き抽選アルゴリズムを用いて着順を決定し、
単勝・複勝の配当を計算します。

## 🧠 アプリ概要

本アプリでは以下の流れでレースを実行します。

1. ユーザーが馬・賭け金・賭け方を選択
2. オッズを重みとした確率抽選で着順を決定
3. 単勝または複勝の配当を計算
4. 所持金へ反映

---

## ⚙️ 技術スタック

- React

- useState による状態管理
- 重み付き確率抽選アルゴリズム
